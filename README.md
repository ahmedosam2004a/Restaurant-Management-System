# Restaurant Management System

## Overview

Restaurant Management System is a desktop application developed using C# Windows Forms and SQL Server. The system helps restaurants manage daily operations, including cashier transactions, table reservations, menu items, and customer orders.

## Features

- User Login
- Cashier (POS)
- Table Reservation
- Table Management
- Menu Item Management
- Order Management
- Invoice Generation
- SQL Server Database Integration

## Technologies Used

- C#
- Windows Forms
- SQL Server
- ADO.NET
- Visual Studio

## Installation

1. Clone the repository.
2. Restore the SQL Server database.
3. Update the connection string.
4. Open the solution in Visual Studio.
5. Run the application.
